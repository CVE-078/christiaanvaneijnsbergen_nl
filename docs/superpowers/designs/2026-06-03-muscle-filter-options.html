Recommendation — Option C, the scroll rail

It keeps the filter to one tap, one level — no menu to open, no second sub-row to learn. That matches how the current chips already work, so it's the smallest behavioural change and the lowest-risk edit. The fixes that matter are real: a single non-wrapping line (kills the cluttered multi-row wrap), a count badge per muscle so empty categories are obvious before you tap, and a fade edge that signals there's more to scroll.

A (dropdown) is the densest and scales best, but it hides every option behind a tap and adds an interaction to a filter people flick through constantly. B (grouped) is elegant but coarser and two-level — overkill for 10 categories, and the Push/Pull/Legs grouping fights the fact that the library is browsed by single muscle. Pick C unless you expect the category list to grow well beyond 10, in which case A ages better.
